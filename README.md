# Endless Runner

> ⚠️ **Work in Progress** – This game is currently under development. Features, controls, and visuals are subject to change.


---

## 📺 Display Support

- Optimized for **16:9** aspect ratio displays

---

## 🛠️ Engine Version

- **Unity 6 LTS** (6.3)

---

## 🎮 Controls

| Action | Key(s) |
|--------|--------|
| Move | `←` `↑` `→` or `WASD` |
| Slide | `Left Shift` |
| Jump | `Space` |
